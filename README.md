# kiva-tietaa
"Kiva tietää näin kolmesta aamyöstä" Hackfest project.

# Development

## Backend

Run mongoDb:

```
docker-compose up
```

Start server:

```
cd blackkari
npm dev run
```

Ping server:

```
curl localhost:3000/ping
```
